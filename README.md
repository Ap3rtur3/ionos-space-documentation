# Deploy Now Documentation

Uses [Gridsome](https://gridsome.org) with NodeJS v12.22

## Setup

Install dependencies and edit new `.env` file
```bash
npm ci
cp .env.example .env
```

Run local server
```bash
npm run develop
```

Create static build in `dist` folder
```bash
npm run build
```
